# Procédure rapide

## 1) GitHub
- Dézipper le projet
- Créer un repo GitHub vide
- Envoyer tous les fichiers du zip

## 2) Codemagic
- Connecter le repo GitHub
- Sélectionner le fichier `codemagic.yaml`
- Lancer le workflow `android-and-web`

## 3) Récupération
L'APK sera disponible dans les artifacts du build Codemagic.
